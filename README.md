# onlineShop_fullStack
Made with @reactjs @nodejs
# Project Title

MERN_Shop

## Description

A project that try to recreate a online shop, with products and manufactures info.

## Getting Started

### Dependencies

- Font Awesome
- Axios
- Boostrap
- env-cmd
- React (v17.0.02)
- react-burguer-menu
- react-data-table-component
- react-router-dom
- sass
- styled-components
- express(v4.17.1)
- node-fetch(v3.0.0)
- Postgress(v8.7.1)
- Firebase(v9.1.2)

### Installing

- git clone (repo url)

- npm i -> on main folder

### Executing program

- Open two terminals, one for backend and one for frontend
- npm start in both
